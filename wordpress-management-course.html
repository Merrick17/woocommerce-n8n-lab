One-Time WordPress Settings for SEO

WP for Non-Coders
  1. Settings → General — correct site title and tagline
  2. Settings → Permalinks — select Post name → Save
  3. Settings → Reading — set homepage (static page or blog)
  4. Settings → Privacy — ensure "Discourage search engines" is OFF on live site
  5. Install SSL — hosting panel → Force HTTPS (most hosts: one click)
Non-coder ruleIf a setting isn't listed above or in your SEO plugin, you probably don't need to touch it.

SEO · AEO · GEO on WordPressModule 3 · Section

Install Rank Math — Click by Click

SEO Plugin Setup
  1. Plugins → Add New → search "Rank Math SEO"
  2. Install Now → Activate
  3. Click Start Wizard — do not skip
  4. Choose site type: Business / Blog / Shop
  5. Connect Google Search Console (optional but recommended)
  6. Enable Sitemap, Schema, Redirection modules
  7. Finish wizard → go to Rank Math → Dashboard
ImportantInstall ONLY Rank Math OR Yoast — never both. They conflict.

The Non-Coder On-Page SEO Checklist (Per Page)

On-Page SEO
  • Focus keyword — enter in Rank Math/Yoast metabox
  • SEO title — compelling, under 60 characters
  • Meta description — 150–155 chars, includes keyword + CTA

Module 5

Technical SEO Without Writing Code

Sitemaps, speed, mobile, HTTPS — all handled by plugins.

SEO · AEO · GEO on WordPressModule 5 · Section

Technical SEO Tasks → Plugin Solution

Technical SEO
  • XML Sitemap → Rank Math / Yoast (automatic)
  • Robots.txt → Rank Math → Edit robots.txt (visual editor)
  • HTTPS/SSL → Hosting panel or Really Simple SSL plugin
  • Page speed → LiteSpeed Cache or WP Rocket (toggle ON)
  • Image compression → ShortPixel or Imagify plugin
  • Broken links → Rank Math 404 Monitor + Redirection plugin
  • Mobile-friendly → choose mobile-responsive theme (Astra, Kadence)
SEO · AEO · GEO on WordPressModule 5 · Slide 25

Speed Plugin Settings — Non-Coder Presets

Technical SEO

Install LiteSpeed Cache (free) OR WP Rocket (paid). Enable these toggles:

  • ✅ Enable Cache
  • ✅ Browser Cache
  • ✅ Lazy Load Images
  • ✅ Minify CSS (test site after — disable if layout breaks)
  • ✅ Minify JavaScript (same — test checkout if WooCommerce)
  • ❌ Exclude: /cart/, /checkout/, /my-account/ from cache
Test after enablingVisit site in incognito. Run PageSpeed Insights (Google, free). Target 80+ mobile score.
